# 《雀趣消除乐》软著材料包使用说明

本目录用于准备《雀趣消除乐游戏软件 V1.0》的个人计算机软件著作权登记材料，并辅助后续抖音小游戏资质准入和备案。

## 文件用途

1. `01-软著申请信息采集表-雀趣消除乐V1.0.docx`
   - 用于汇总在线申请所需信息。
   - 请自行填写姓名、身份证号码、手机号、邮箱、通讯地址、邮编和签字日期。
   - 请按实际设备补充开发硬件和 Windows 版本。

2. `02-软件说明书-雀趣消除乐V1.0.docx`
   - 已根据当前项目代码整理功能和操作流程。
   - 正式提交前，请将实际运行截图插入“界面截图补充清单”一节，并删除清单提示文字。
   - 截图应至少覆盖：首页、对局主界面、消除反馈、设置界面、关卡结果。

3. `03-源程序鉴别材料-雀趣消除乐V1.0.docx`
   - 已整理项目中 5 个自有 TypeScript 源文件，共 2965 行、60 页。
   - 不包含第三方库、缓存、构建产物、账号或密钥。
   - 正式提交前请确认办理渠道对源程序页数、每页行数和格式的最新要求。

4. `04-提交前核对清单-雀趣消除乐V1.0.docx`
   - 按软著申请、证书领取、抖音资质准入和备案阶段逐项核对。

## 重要提示

- 抖音小游戏名称和游戏内展示名称应使用“雀趣消除乐”，并与软著名称或简称保持一致。
- 若抖音平台主体不是软著登记的个人著作权人，需要按平台要求另行准备运营/发行授权证明。
- 电子软著可用于平台资质准入，但小游戏备案应准备国家版权局发放的软著登记证书。
- 游戏中的字体、音乐、音效、美术和其他素材如来自第三方，应保留合法授权或购买凭证。
- 本材料包是申报准备文件，不替代登记机构或平台的最终审核意见；提交前请以办理页面的最新要求为准。
