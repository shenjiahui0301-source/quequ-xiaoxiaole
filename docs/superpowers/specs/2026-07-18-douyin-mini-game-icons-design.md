# 抖音小游戏图标设计规格

## 目标

为《雀趣消除乐》制作 3 个可供筛选的抖音小游戏图标。图标延续项目现有的可爱 Q 版麻将美术风格，并在抖音小游戏列表的小尺寸展示环境中保持清晰辨识。

## 交付规格

- 数量：3 个独立候选图标。
- 格式：PNG。
- 尺寸：每张 600×600 像素。
- 画布：完整正方形，四角不做圆角或透明圆角。
- 禁止内容：水印、二维码、平台标识、游戏名称及其他文字。
- 安全构图：核心主体集中在画面中央区域，四周保留适量呼吸空间，避免平台裁切影响识别。

## 统一视觉语言

- 暖黄色、金黄色为主背景，搭配绿色描边和红、蓝、绿麻将花色。
- 采用圆润、明亮、轻立体的休闲游戏插画质感。
- 使用高光、星芒和轻微爆破元素表现轻松、爽快的消除反馈。
- 麻将牌边缘清楚、主体轮廓简洁，确保缩小到应用列表尺寸后仍可识别。
- 延续现有图标的萌趣氛围，但不沿用其圆角画布。

## 三个候选方向

### 方案 A：眨眼萌牌（主推）

一张拟人化麻将牌占据视觉中心，带有眨眼或开心表情；左右可配少量圆形麻将花色或消除光效。该方案主体最大、角色感最强，适合作为品牌主图标。

#### 用户筛选后的修订

保留方案 A 的暖黄色爆发背景、绿色描边、Q 版轻立体质感和眨眼主角，将单牌构图改为双牌“主角＋伙伴”构图：原眨眼麻将位于前方并保持最大视觉权重；第二张带不同圆点花色的笑脸麻将从后方斜向露出。两张牌轻微靠拢形成互动，主体依然集中、轮廓清楚，不能退化为平均并排，也不得出现文字、圆角、水印或二维码。

### 方案 B：三牌消除

三张麻将牌以轻微错位或碰撞姿态组成紧凑三角构图，中心出现明亮星芒和小型消除粒子。该方案更直观地表达“三消”玩法与爽快反馈。

### 方案 C：雀趣组合

多张经典麻将牌采用扇形或向上绽放式排布，通过红、蓝、绿花色形成丰富层次。该方案题材信息最充分，氛围热闹，强调麻将主题。

## 验收标准

1. 三张图均为精确的 600×600 PNG。
2. 四角为正常方形画面内容，不存在圆角、透明圆角或圆角遮罩。
3. 图中无水印、二维码、游戏名称、平台标识和乱码文字。
4. 三张图具有同一套美术语言，同时在主体构图上明显不同。
5. 缩略展示时能迅速识别为可爱麻将消除类游戏。
